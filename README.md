
# ENVS-193DS_homework-03
Homework 03 for ENVS-193DS: Soil Salinity and Personal Data Analysis.

## General Information
This repository contains the data, code, and visualizations for Homework 3. The project includes a statistical critique of an agricultural study and a personal data project tracking the relationship between sleep, motivation, and physical performance.

## Data and File Information
* **Data/**: Contains `salinity-pickleweed.csv` and `personal_pushup_data.csv`.
* **Code/**: Contains `homework-03.qmd`, the Quarto file used for analysis.
